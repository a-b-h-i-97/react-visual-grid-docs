# Thumbnail sizes

You can specify the size of the thumbnails via the [imageSizes](../props.md#imagesizes) prop.The imageSizes prop is an array of objects that defines the dimensions of the image.

The user can alter the size of the thumbnails using the control bar at the bottom. There are three thumbnail sizes available: 1x, 2x, and 3x.

The library includes a set of thumbnail sizes by default.

```jsx
{
  "1X": {
    width: 120,
    height: 100,
  },
  "2X": {
    width: 200,
    height: 180,
  },
  "3X": {
    width: 320,
    height: 280,
  },
};
```

You can override the default set by passing your own array of objects to the [imageSizes](../props.md#imagesizes) prop.

```jsx title="App.jsx"
import { Grid } from "react-visual-grid";

const App = () => {
  return (
    <Grid images={images} gridLayout="vertical" width={1800} height={1200} imageSizes={{
      "1X": {
        width: 120,
        height: 100,
      },
      "2X": {
        width: 200,
        height: 180,
      },
      "3X": {
        width: 320,
        height: 280,
      },
    }} />
  );
};
```
